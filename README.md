# 汽车理论车速计算器

一个简约的 H5 单页应用，用于计算汽车理论车速。

## 在线使用

https://joxin-1.github.io/car-speed-calculator/

微信扫码或点击链接即可使用，无需跳转外部浏览器。

## 输入参数

| 参数 | 单位 | 说明 |
|------|------|------|
| 转速 | r/min | 发动机转速 |
| 轮毂尺寸 | inch | 轮毂直径 |
| 轮胎宽度 | mm | 如 235/55R18 中 235mm |
| 扁平比 | % | 如 55 表示 55%，填 55 |
| 变速箱传动比 | — | 当前挡位传动比 |
| 主减速比 | — | 终传动比 |

## 公式

`
V = [0.377 × n × (d × 1.27 + B_cm × k)] / (i₁ × i₂ × 100)
`

- n = 转速 (r/min)
- d = 轮毂尺寸 (inch)
- B_cm = 轮胎宽度 (mm) ÷ 10 → cm
- k = 扁平比 ÷ 100 → 小数
- i₁ = 变速箱传动比
- i₂ = 主减速比

结果单位：km/h

## 功能

- 计算理论车速，保留 2 位小数
- 保存近 5 次计算记录（localStorage）
- 点击历史记录可复用参数
- 简约界面，适配微信内置浏览器
- 支持浅色/深色模式
